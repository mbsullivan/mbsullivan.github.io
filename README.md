Personal academic website for Michael B. Sullivan.

Built with Mako templates (via www-pysub2) and served as static HTML.

## Structure

| Directory              | Contents                                            |
|------------------------|-----------------------------------------------------|
| `source/`              | Mako templates for each page                        |
| `source/includes/`     | Shared base templates (header, footer, nav)         |
| `source/research_bib/` | BibTeX bibliography and parsing logic (`pybib.py`)  |
| `structure/`           | CSS, JS, fonts, and static assets                   |
| `attachments/`         | PDFs (CV, resume, papers), images, posters, presentations |
| `*.html`               | Generated output (do not edit directly)             |
